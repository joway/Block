# Block

> Another blog based on Django . 

===

## Usage

### Env

- Python 3.5.1
- Mysql
- Redis
- [Nginx]

### Install

	git clone 
	
	cd Block 
	
	pip install -r requirements.txt 
	
	python manage.py runserver 0.0.0.0:8000
	
### Custom

	cp prov_settings.py local_settings.py
	
	change env in local_settings.py
